Metode sistem pakar

Certainty Factor untuk menghitung tingkat keyakinan

Certainty Factor digunakan untuk mengukur seberapa kuat gejala yang dipilih pengguna mendukung suatu penyakit. Pada sistem ini, nilai CF pakar dibentuk dari frekuensi gejala pada dataset THT.

Bobot gejala

CF pakar dari dataset

CF_pakar = jumlah data penyakit dengan gejala / total data penyakit

Contoh: jika gejala nyeri telinga muncul pada 53,4% data Otitis Media, maka CF pakar untuk gejala tersebut adalah 0,534.

Input user

Checklist gejala

CF_gejala = CF_pakar x CF_user

Karena input berupa checklist, sistem memakai CF_user = 1 untuk gejala yang dipilih pengguna.

Kombinasi CF

Rumus penggabungan beberapa gejala

1

Pilih gejala

User mencentang gejala yang dialami pasien pada form diagnosis.

2

Cocokkan aturan

Sistem mencocokkan gejala dengan basis aturan masing-masing penyakit THT.

3

Gabungkan CF

CF_combine = CF_old + CF_new x (1 - CF_old)

4

Tampilkan ranking

Penyakit dengan nilai CF tertinggi menjadi hasil diagnosis utama.
